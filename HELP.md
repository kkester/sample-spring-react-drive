### Run MySQL in Docker
```
docker run --name=mysql-db -e MYSQL_ROOT_PASSWORD=admin -e MYSQL_DATABASE=placesdb -p3306:3306 -d mysql/mysql-server:latest
docker container exec -it mysql-db bash
mysql -uroot -padmin
```
